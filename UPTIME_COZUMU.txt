# Discord Bot 7/24 Aktif Tutma Çözümü

Bu dosya, Discord botunuzu Replit üzerinde tamamen ücretsiz ve otomatik olarak 7/24 aktif tutmak için hazırlanmış çözümü açıklar.

## Çözüm Bileşenleri

1. **keep-bot-online.js**: 
   - Anında HTTP server başlatır (port 3500)
   - Belirtilen URL'ye düzenli ping atar (5 dakikada bir)
   - Ping başarısı/başarısızlığını loglar
   - Durum bilgisini JSON dosyasına kaydeder
   - Her dakika disk aktivitesi yaratır (Replit'in uyku moduna geçmesini engeller)

2. **ping-target-bot.js**:
   - Kullanıcının istediği URL'yi düzenli ping atar (5 dakikada bir)
   - Özel parametreler ekleyerek cache sorunlarını önler
   - Ayrı bir log dosyasına sonuçları kaydeder

3. **public/always-online.html**:
   - UptimeRobot için optimize edilmiş HTML sayfası
   - Dinamik içerik üretir (her yüklemede farklı değer)
   - Cache önleyici meta tag'ler içerir
   - Basit ve etkili "ping endpoint" olarak çalışır

## Kullanım Adımları

1. Terminal ekranında şu komutları çalıştırın:
   ```
   node keep-bot-online.js
   ```

   Ayrı bir terminal ekranında:
   ```
   node ping-target-bot.js
   ```

2. UptimeRobot'a gidip yeni "HTTP(s)" tipinde bir monitor ekleyin:
   - URL: https://discord-halisaha-manager.emilswd.repl.co/always-online.html
   - Interval: 5 dakika
   - NOT: Monitor tipi olarak "PING" DEĞİL, "HTTP(s)" seçin!

3. Replit'in "Always On" özelliğini açın:
   - Run > Always On (Enable)

## Önemli Notlar

- Bu çözüm, birden fazla servis kullanarak tam yedekli uptime sağlar
- Discord botunuzun internet bağlantınız olmadığında bile çalışmaya devam etmesini sağlar
- Chrome tarayıcısını kapatsanız bile bot çalışmaya devam eder
- Günlük olarak "Run" butonuna bir kez tıklamak idealdir (zorunlu değil)
- UptimeRobot'ta "PING" yerine mutlaka "HTTP(s)" monitor tipi kullanın
- Ping sorunu yaşarsanız başka bir uptime servisi (Cron-Job.org) de deneyebilirsiniz
